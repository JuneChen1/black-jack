# Black Jack ( 21點 )

+ 遊戲開始時，玩家獲得兩張牌，莊家獲得一張明牌
+ J、Q、K 為 10 點，A 可為 1 點或 11 點，2~10 點以牌面點數計算
+ 玩家可加牌或停止要牌，注意當點數大於 21 點時，即為爆牌，此時莊家勝利
+ 如玩家未爆牌且停止要牌，換莊家加牌，莊家點數不足 17 點時，必須要牌
+ 如莊家點數大於 21 點，即為爆牌，此時玩家勝利
+ 雙方皆未爆牌的情況下，點數大者勝利
