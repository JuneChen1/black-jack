# Black Jack ( 21點 )

+ 遊戲開始時，玩家進行下注
+ 玩家獲得兩張牌，莊家獲得一張明牌
+ J、Q、K 為 10 點，A 可為 1 點或 11 點，2~10 點以牌面點數計算
+ 當拿到兩張相同點數的牌時，可選擇是否分牌
+ 玩家可選擇加注或投降，投降收回一半籌碼(前一項選擇分牌則不可投降)
+ 玩家可加牌或停止要牌，注意當點數大於 21 點時，即為爆牌，此時莊家勝利
+ 如玩家未爆牌且停止要牌，換莊家加牌，莊家點數不足 17 點時，必須要牌
+ 如莊家點數大於 21 點，即為爆牌，此時玩家勝利
+ 雙方皆未爆牌的情況下，點數大者勝利
+ 可持續遊玩，並統計雙方勝利的次數與玩家籌碼

<br>

### 特殊情狀：Black Jack
即為開場拿到 A + 10/J/Q/K 牌，勝負分為三種情況：
1. 玩家 Black Jack 且莊家沒有 => 玩家勝利
2. 莊家 Black Jack 且玩家沒有 => 莊家勝利
3. 雙方 Black Jack => 和局
